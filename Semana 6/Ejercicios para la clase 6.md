## Ejercicios para la clase 6

### Sistema Cliente-Servidor para Gestión de Notas 

Imagina que estás creando una aplicación sencilla para tomar notas usando la arquitectura cliente-servidor. En este sistema, el cliente será la parte que muestra las notas y permite al usuario agregar nuevas. 

El servidor se encarga de almacenar las notas y enviar las respuestas al cliente cuando este las solicite. 

### Requerimientos: 

**1.** Cliente: El usuario podrá ver sus notas, agregar una nueva nota o eliminar una nota existente. El cliente enviará estas solicitudes al servidor. 

**2.** Servidor: El servidor almacenará las notas y devolverá la información al cliente cuando este lo pida. 

### Tarea 

**1.** Describe brevemente qué hace el cliente y qué hace el servidor en este sistema. 

**2.** Menciona un beneficio de usar la arquitectura cliente-servidor. 

**3.** Identifica un desafío que podrías enfrentar usando este modelo.
