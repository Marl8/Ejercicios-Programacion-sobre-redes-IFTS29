## Consigna Ejercicio Semana 4

### Sistema Simple de Preguntas y Respuestas Cliente-Servidor

Imagina que estás creando un sistema simple de preguntas y respuestas usando el modelo cliente-servidor. En este sistema, el cliente es un usuario que envía preguntas al servidor, y el servidor procesa las preguntas y responde con la respuesta adecuada. 


### Requerimientos

1. El cliente envía una pregunta al servidor. 

2. El servidor recibe la pregunta, la procesa, y envía una respuesta al cliente. 

**Flujo de trabajo:**

- El cliente pregunta, por ejemplo, "¿Cuál es la capital de Francia?". 

- El servidor responde con la información: "París". 


### Tarea 

1. Explica cómo el cliente envía la pregunta al servidor y cómo el servidor devuelve la respuesta al cliente.

2. Menciona una ventaja de usar el modelo cliente-servidor en este sistema (por ejemplo, centralización de información). 

3. Menciona una desventaja (por ejemplo, si el servidor se cae, el cliente no podrá obtener respuestas).
